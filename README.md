## tcp是Go语言实现TCP通信
如果没有tcp/client和tcp/server项目，则按以下方式创建。
### 创建tcp/client客户端
1. 创建tcp目录
2. 在tcp目录创建client
3. client目录下执行go mod init client命令生成go.mod文件

### 创建tcp/server服务端
1. 进入tcp目录
2. 在tcp目录创建server
3. server目录下执行go mod init server命令生成go.mod文件
